## 🚀 使用技巧

### 快速找到需要的工具

1. **使用搜索框**
   - 输入工具名称
   - 输入功能关键词
   - 输入标签

2. **选择分类**
   - 点击顶部分类标签
   - 快速定位到对应类别

3. **高级筛选**
   - 勾选"免费工具"只看免费
   - 勾选"中文工具"只看支持中文的
   - 按热度或评分排序

### 查看工具详情

**网格视图**：适合快速浏览
- 显示工具图标、名称、简介
- 显示标签、评分、价格
- 一键访问官网

**列表视图**：适合查看详情
- 显示完整描述
- 显示所有标签
- 更详细的信息
