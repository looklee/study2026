## 🚀 使用教程

### 1. 上传文档

**步骤**:
1. 点击"上传文档"按钮
2. 填写文档信息
   - 文档名称
   - 文档内容（粘贴文本）
   - 分类
   - 标签
3. 点击"上传"

**自动处理**:
- 文本自动分块（500 字符/块）
- 生成向量嵌入
- 建立搜索索引

### 2. 搜索文档

**简单搜索**:
1. 在搜索框输入关键词
2. 按回车或点击"搜索"
3. 查看搜索结果（按相关性排序）

**分类筛选**:
1. 点击分类标签
2. 查看该分类下的文档

### 3. RAG 查询

**API 调用**:
```bash
POST /api/v1/knowledge/rag-query
Content-Type: application/json

{
  "query": "什么是机器学习？",
  "limit": 3
}
```

**响应**:
```json
{
  "status": "success",
  "query": "什么是机器学习？",
  "context": "[来源 1] 机器学习笔记：机器学习是 AI 的一个分支...",
  "sources_count": 3,
  "sources": [...]
}
```

## 🔍 搜索算法

### 相关性评分
1. **标题匹配** - +10 分
2. **标签匹配** - +5 分
3. **内容完全匹配** - +3 分
4. **关键词匹配** - +0.5 分/词

### 搜索示例
```
查询："机器学习"

匹配结果:
1. 《机器学习笔记》- 标题匹配 (10 分)
2. #机器学习 标签 - 标签匹配 (5 分)
3. 内容包含"机器学习" - 内容匹配 (3 分)
```

## 📝 文本分块

### 分块策略
- **块大小**: 500 字符
- **重叠**: 50 字符
- **目的**: 保持上下文连贯

### 示例
```
原文：[0-1000 字符的文档]

分块后:
块 1: [0-500]
块 2: [450-950]  (重叠 50)
块 3: [900-1000]
```

## 🐛 故障排除

### 问题 1: 文档上传失败
**解决**: 检查内容是否为空，分类是否有效

### 问题 2: 搜索无结果
**解决**: 尝试不同关键词，检查文档状态是否为"已处理"

### 问题 3: 文档列表为空
**解决**: 先上传文档，刷新页面
