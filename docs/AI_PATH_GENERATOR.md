# 🎯 AI 生成学习路径使用指南

## ✅ 功能说明

`/paths/new` 页面现已接入真实的 AI 功能，使用你配置的 API（Qwen/OpenAI/Anthropic）生成个性化学习路径。

---

## 🚀 使用流程

### 1. 配置 AI API（首次使用）

**必须先配置 API 才能使用 AI 生成功能**

1. 访问 http://localhost:3000/integrations
2. 选择 "通义千问 (Qwen)" 或其他 AI 提供商
3. 点击 "配置" 按钮
4. 输入 API Key
5. 保存并测试

**获取 Qwen API Key**:
- 访问：https://dashscope.console.aliyun.com/apiKey
- 新用户送 ¥18 体验金

### 2. 填写学习信息

访问 http://localhost:3000/paths/new

填写表单：
- **当前水平**：初学者/中级/高级
- **学习目标**：例如 "掌握机器学习基础"
- **每周可用时间**：例如 10 小时
- **学习风格**：混合式/动手实践/理论学习/视频教程
- **已有经验**：例如 "Python 编程，线性代数"

### 3. AI 生成路径

点击 "AI 生成学习路径" 按钮

**AI 将生成**：
- ✅ 个性化学习路径名称
- ✅ 分阶段学习计划（4 个阶段）
- ✅ 每个阶段的目标和主题
- ✅ 每周学习安排
- ✅ 学习建议和提示

### 4. 查看和编辑

生成完成后：
- 自动保存到本地存储
- 跳转到 `/paths` 页面
- 点击路径查看详情
- 可以手动编辑和调整

## 🔧 API 调用流程

```
用户填写表单
    ↓
点击生成按钮
    ↓
前端调用 /api/v1/chat/message
    ↓
后端读取 api_keys.json
    ↓
使用 Qwen API 生成内容
    ↓
解析 JSON 响应
    ↓
保存到 localStorage
    ↓
跳转到 /paths 页面
```

### API 调用示例

**请求**:
```javascript
POST http://localhost:8001/api/v1/chat/message
Content-Type: application/json

{
  "message": "请生成学习路径 JSON：...",
  "system_prompt": "你是一位专业的 AI 学习路径规划专家..."
}
```

**响应**:
```json
{
  "message": "{...JSON 格式的学习路径...}",
  "provider": "qwen",
  "conversationId": "conv_123456"
}
```

## ⚠️ 常见问题

### 问题 1: 提示"请先配置 API 密钥"

**解决**:
1. 点击提示中的"去配置 API"链接
2. 或访问 `/integrations` 页面
3. 配置至少一个 AI API

### 问题 2: AI 生成失败

**解决**:
1. 检查 API Key 是否正确
2. 检查 API 余额是否充足
3. 查看浏览器控制台错误信息
4. 尝试刷新页面重新生成

### 问题 3: 生成的内容不是 JSON 格式

**解决**:
- 系统会自动尝试提取 JSON
- 如果提取失败，会生成默认路径
- 可以在控制台查看详细响应

### 问题 4: 生成速度很慢

**说明**:
- AI 生成需要 5-15 秒
- 显示"AI 正在生成学习路径..."动画
- 请耐心等待，不要关闭页面

## 🎯 最佳实践

### 填写技巧

1. **学习目标要具体**
   - ✅ 好："掌握 Python Web 开发，能独立建站"
   - ❌ 差："学习编程"

2. **时间评估要实际**
   - 考虑工作/学习时间
   - 预留缓冲时间
   - 宁可少估不要多估

3. **已有经验要详细**
   - 列出相关技能
   - 包括自学过的内容
   - 注明掌握程度

### 生成后

1. **检查路径合理性**
   - 阶段划分是否合理
   - 时间分配是否合适
   - 内容是否符合目标

2. **可以手动调整**
   - 修改阶段名称
   - 调整时间分配
   - 添加/删除主题

3. **保存到路径列表**
   - 自动生成后自动保存
   - 在 `/paths` 页面查看
   - 随时可以继续编辑
