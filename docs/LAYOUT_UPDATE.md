## 📐 布局结构

每个页面独立
├── 页面 1 (包含侧边栏)

1. **点击导航项** - 跳转到对应页面

- 侧边栏内容过多时可滚动
- 主内容区独立滚动
- 互不干扰

## 🔧 自定义

### 修改导航项

编辑 `apps/web/src/components/Sidebar.tsx`:

```typescript
const navigation = [
  { name: '新页面', href: '/new-page', icon: NewIcon },
  // ... 现有导航
]
```

### 修改颜色

编辑 CSS 变量或 Tailwind 类：

```css
:root {
  --sidebar-bg: #111827;  /* 自定义侧边栏背景 */
  --sidebar-active: #2563eb;  /* 自定义激活颜色 */
}
```

### 添加分组

在侧边栏中添加分组标题：

```tsx
<div className="mt-8 pt-4 border-t border-gray-800">
  <p className="px-3 text-xs font-semibold text-gray-500 uppercase">
    分组名称
  </p>
  {/* 分组导航项 */}
</div>
```
