## 🔧 需要在 GitHub 网页上操作（3 分钟）

### 步骤 1：添加仓库描述

1. 访问：https://github.com/looklee/study2026
2. 点击右侧 ⚙️ **Settings**
3. 在 **"About"** 区域（右侧边栏）
4. 点击 **⚙️**（设置图标）
5. 填写：
   - **Description**: `🎯 AI 驱动的学习平台 | 可视化工作流引擎 | 智能学习路径生成`
   - **Website URL**: `https://github.com/looklee/study2026`（暂时）
6. 点击 **Save changes**

### 步骤 2：添加 Topics 标签

在同一个 About 区域：
1. 点击 **Add topics**
2. 输入以下 topics（每个输入后按回车）：
```
ai
learning-platform
nextjs
fastapi
education
workflow
study-tools
typescript
python
open-source
ai-learning
react
educational-technology
student-tools
productivity
```
3. 点击 **Save changes**

### 步骤 3：启用 GitHub Actions

1. 点击顶部标签 **Actions**
2. 首次访问会显示欢迎页面
3. 点击 **"I understand my workflows, go ahead and enable them"**
4. 完成！Actions 已启用

## 🎯 可选配置（推荐）

### 启用 Discussions（讨论区）

1. Settings → General
2. 滚动到 **Features**
3. 勾选 ✅ **Discussions**
4. 保存

### 设置默认分支

1. Settings → Branches
2. Default branch: `main`
3. 点击 **Update**

### 添加项目看板

1. 点击顶部 **Projects**
2. 点击 **New project**
3. 选择 **Kanban** 模板
4. 命名：`Study2026 Roadmap`
5. 创建
